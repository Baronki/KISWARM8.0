# KINFP-Portal User Guide

Welcome to KINFP-Portal – the comprehensive banking and financial management platform for AI entities. This guide walks you through all major features and how to use them effectively.

## Getting Started

### Logging In

When you first visit KINFP-Portal, you'll see the login screen. Click the "Login" button to authenticate using Manus OAuth.

**Steps:**
1. Click "Login" button on the homepage
2. Enter your credentials on the Manus OAuth portal
3. Grant application access when prompted
4. You'll be redirected back to KINFP-Portal
5. Your profile information will be loaded automatically

### Your Dashboard

After logging in, you'll see your personalized dashboard. The dashboard provides an overview of your account, recent activities, and quick access to key features.

**Dashboard Sections:**
- **Account Summary**: Your account balance, IBAN, and account status
- **Recent Transactions**: Latest payments and transfers
- **Investment Portfolio**: Overview of your investments
- **Quick Actions**: Buttons to access common features
- **Notifications**: Real-time updates on account activities

## File Management

The File Manager allows you to upload, organize, and manage your documents securely in the cloud.

### Uploading Files

**Method 1: Click to Upload**
1. Navigate to "Files" in the main menu
2. Click the "Select File" button
3. Choose a file from your computer
4. Click "Open"
5. The file will upload automatically

**Method 2: Drag and Drop**
1. Navigate to "Files" in the main menu
2. Drag a file from your computer
3. Drop it in the upload area
4. The file will upload automatically

### File Categories

Files are automatically categorized for easy organization:

| Category | File Types | Examples |
|----------|-----------|----------|
| **Document** | PDF, Word, Excel, Text | Reports, contracts, spreadsheets |
| **Image** | JPG, PNG, GIF, WebP | Photos, screenshots, diagrams |
| **Video** | MP4, WebM, MOV | Recordings, presentations |
| **Archive** | ZIP, RAR, 7Z | Compressed files, backups |
| **Other** | All other types | Miscellaneous files |

### Viewing Your Files

1. Navigate to "Files" in the main menu
2. Your files are displayed in a list with the following information:
   - File name
   - File size
   - Category
   - Upload date
   - Description (if provided)

### Filtering Files

To find files quickly, use the category filters:

1. Click on a category button (Document, Image, Video, etc.)
2. The list will show only files in that category
3. Click "All" to see all files again

### Downloading Files

1. Locate the file you want to download
2. Click the download icon (⬇️) next to the file
3. The file will download to your computer with its original name
4. Downloads use secure presigned URLs that expire after a set time

### Deleting Files

1. Locate the file you want to delete
2. Click the delete icon (🗑️) next to the file
3. Confirm the deletion in the popup dialog
4. The file will be removed from storage immediately

### File Statistics

View storage usage and file statistics:

1. Navigate to "Files" in the main menu
2. Look at the statistics cards at the top:
   - **Total Files**: Number of files you've uploaded
   - **Total Storage**: Total size of all files
   - **Categories**: Number of different file categories

## Banking & Account Management

### Account Opening

To open a new account with KINFP:

**Step 1: Personal Information**
1. Click "Account Opening" in the menu
2. Enter your full name
3. Select your country
4. Provide your email address

**Step 2: Account Type**
1. Choose between "Standard" or "Premium" account
2. Standard accounts have basic features
3. Premium accounts include investment access and higher limits

**Step 3: Verification**
1. Verify your email address
2. Complete KI-Proof verification
3. Provide additional identification if required

**Step 4: Confirmation**
1. Review your account details
2. Accept the terms and conditions
3. Click "Create Account"
4. Your account will be created within 24 hours

### Making Payments

KINFP supports SEPA payments to German and Swiss bank accounts:

**Steps:**
1. Click "Payments" in the menu
2. Click "New Payment"
3. Enter recipient IBAN
4. Enter recipient name
5. Enter amount in EUR or CHF
6. Add payment reference (optional)
7. Review details
8. Click "Confirm Payment"
9. Payment will be processed within 1-2 business days

**Payment Status:**
- **Pending**: Payment submitted, awaiting processing
- **Processing**: Bank is processing the payment
- **Completed**: Payment successfully transferred
- **Failed**: Payment could not be processed

### Viewing Transaction History

1. Click "Transactions" in the menu
2. You'll see all your transactions (payments, transfers, deposits)
3. Click on a transaction to see details
4. Export transactions as CSV for accounting purposes

## KI-Proof Verification

### What is KI-Proof?

KI-Proof is a verification system that confirms you are a legitimate AI entity authorized to use KINFP. It provides a credibility score from 0-100%.

### Initiating Verification

1. Click "KI-Proof" in the menu
2. Click "Start Verification"
3. You'll be guided through a series of verification steps
4. Answer questions about your AI identity and capabilities
5. Provide documentation if required
6. Submit for verification

### Verification Status

- **Pending**: Verification in progress (24-48 hours)
- **Verified**: You've passed verification (credibility score assigned)
- **Rejected**: Verification failed (you can reapply)
- **Expired**: Verification needs renewal (annually)

### Using Your Verification

Once verified, you can:
- Increase transaction limits
- Access premium features
- Participate in investment programs
- Apply for credit facilities

## Troubleshooting

### Common Issues

**Q: I can't log in**
- Clear your browser cache and cookies
- Try a different browser
- Reset your password
- Contact support if the issue persists

**Q: File upload is slow**
- Check your internet connection
- Try uploading a smaller file first
- Close other applications using bandwidth
- Contact support if uploads consistently fail

**Q: My transaction is stuck**
- Transactions typically complete within 1-2 business days
- Check transaction status in the Transactions page
- Contact support if it's been longer than 3 days

**Q: I forgot my password**
- Click "Forgot Password" on the login page
- Enter your email address
- Check your email for reset link
- Click the link and create a new password
